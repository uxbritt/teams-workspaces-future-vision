Pillar 1 of 4

Hierarchical Navigation

Parent → Region → Location tree that scales to 500+ units

User Story

Franchise Owner

"I want to quickly navigate between my 200+ locations without logging in/out, so I can manage marketing across all units efficiently."

Key Benefits

  • Fast context switching without impersonation
  • Folder-based organization by region
  • Scales beyond flat dropdown limitations (~50 units)
  • Clear visual hierarchy of org structure

Job to be Done

When I'm managing multiple franchise locations, I want to navigate to any workspace instantly, so I can complete my tasks without friction.

Pillar 2 of 4

Roll-up Reporting

Aggregate metrics across all locations without switching

User Story

Regional Marketing Director

"I want to see campaign performance across all my locations in one view, so I can identify top performers and those needing support."

Key Benefits

  • Org-level dashboards with aggregated metrics
  • Performance leaderboards across locations
  • Identify struggling locations quickly
  • Cross-workspace campaign views

Job to be Done

When I'm reviewing marketing performance, I want to see all locations compared side-by-side, so I can allocate resources where they're needed most.

Mockup — Org-level dashboard with location comparison

Pillar 3 of 4

Brand & Access Control

Role-based permissions, brand assets, approval workflows

User Story

Brand Manager

"I want to distribute approved templates and assets to all locations, so franchisees stay on-brand without creative freedom to deviate."

Key Benefits

  • Brand kit sharing to all workspaces
  • Campaign approval before send
  • Template locking and restrictions
  • Role-based permission controls

Job to be Done

When I'm protecting the brand, I want to control what franchisees can modify, so we maintain consistency across all customer touchpoints.
