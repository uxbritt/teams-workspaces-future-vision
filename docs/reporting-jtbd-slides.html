Current State

The Reporting Problem

What's broken for multi-location customers today

No Roll-up Visibility

Must impersonate each workspace individually to check campaign performance. No aggregate view.

Time-Consuming Process

Checking 25+ locations means 25+ login switches. Hours wasted on manual data gathering.

No Comparisons

Can't compare workspace performance side-by-side. No way to identify top or low performers.

Manual Exports

Leadership reports require exporting data from each account and combining in spreadsheets.

Users

Who Needs Reporting?

Three distinct user types with different reporting needs

O

Org Admin

Marketing Director, VP level

  • • See all locations at once
  • • Compare regional performance
  • • Export for leadership
  • • Identify struggling locations
R

Regional Manager

Oversees 5-15 locations

  • • See their region's workspaces
  • • Compare locations in region
  • • Drill into problem areas
  • • Report up to Org Admin
W

Workspace User

Local marketer, single location

  • • See their own campaigns
  • • Track send performance
  • • Compare to previous periods
  • • No need for cross-workspace
