User Testing

"Hub" vs "Organization"

Comparing hierarchy term effectiveness

Recommendation

Use "Organization" for top-level entity. Avoid "Hub" — ambiguous (used for multiple levels) and conflicts with HubSpot mental models.

Confusion Points

elite314

"Now I know that if I go and click into a different hub, it'll take me to different workspaces."

→ Used "Hub" to mean mid-level folders (Northeast Hub), not organization

elite314

"I would explain a hub as a central location... the hub will be named after that and it could have multiple workspaces within the hub."

→ Conflated regional folders with organization-level concept

honeypot13

"I think the difference between hub and workspace in my mind is going to be Hub is like my command center."

→ Initially called it "hub", then used "organization" interchangeably when navigating

elite314

"Was it clear you were moving to a mid-level hub? I'm not sure what mid-level means so no."

→ "Hub" terminology unclear when describing hierarchy levels

Summary

Terminology Recommendations

Based on market research + user testing

Key Findings

• Workspace: Well-understood for individual accounts

• Organization: Clear parent-level hierarchy

• Dashboard: Universal for overview pages

• Hub: Ambiguous — users mixed it with folders
