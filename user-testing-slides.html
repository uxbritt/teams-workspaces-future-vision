Friction Points

Areas for Improvement

Pain points and confusion observed during testing

Folder vs Workspace

Users confused about when to use folders vs workspaces for organization

→ Need clearer onboarding and contextual help

Navigation Depth

Some users got lost when drilling into nested folders

→ Breadcrumbs and "Back to Org" helped when discovered

Notification Visibility

Some top bar icons weren't immediately noticed

→ Consider more prominent placement or badges

Reporting Filters

Users wanted more intuitive date range and workspace filtering

→ Pre-populated filters when linking from dashboard

Onboarding

First-Time Experience

Insights from the signup and onboarding flow

Skip Setup Option

Users appreciated being dropped into the product with a pre-created workspace rather than forced setup steps

Tooltip Guidance

Pendo-style tooltips pointing to the first workspace helped users understand the concept quickly

Empty State Confusion

When folders are empty, users weren't sure what action to take first

Recommendation

Add clear CTAs in empty states: "Invite team members" or "Create first campaign"
